**Session Timeout Hook:**
1. Create `src/hooks/use-session-timeout.ts` based on RESEARCH.md Pattern 6:
   - Track user activity via mousedown, keydown, scroll, touchstart events
   - WARNING_BEFORE_MS = 5 * 60 * 1000 (5 minutes before expiry, per CONTEXT.md)
   - Default session timeout: 8 hours (28800000 ms) -- accept as parameter
   - Set warning timer at (timeout - 5min), expire timer at timeout
   - On activity: reset both timers
   - On expire: redirect to /login?expired=true
   - extendSession(): calls `fetch("/api/auth/session", { method: "GET", credentials: "include" })` to refresh server session, then resets timers
   - Returns: { showWarning: boolean, extendSession: () => Promise<void>, remainingTime: number }
   - remainingTime counts down every second when warning is shown (for display in modal)
