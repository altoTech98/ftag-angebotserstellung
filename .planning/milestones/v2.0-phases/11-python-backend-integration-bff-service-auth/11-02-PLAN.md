<objective>
Create the Next.js BFF proxy layer that forwards all /api/backend/* requests to the Python backend with service authentication and user context headers. Add SSE token issuance for direct browser-to-Python streaming, an SSE client with polling fallback, and a health check endpoint.

Purpose: The browser must never call the Python backend directly for CRUD operations. All requests go through Next.js API routes which add the service key and user identity. SSE streaming connects directly to Python using short-lived tokens issued by Next.js.

Output: BFF catch-all proxy, SSE token route, SSE client library, health check, env vars, passing tests.
