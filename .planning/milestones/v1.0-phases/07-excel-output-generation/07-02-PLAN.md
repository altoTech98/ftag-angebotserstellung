2. Add POST `/api/offer/generate` endpoint:
```python
@router.post("/offer/generate")
async def v2_generate_result(request: GenerateV2ResultRequest):
```
- Lazy import `_analysis_results` from `v2.routers.analyze_v2` (with try/except guard for graceful degradation if v2 not available).
- Validate analysis_id exists in `_analysis_results`, return 404 if not found.
- Create job via `create_job()`, call `run_in_background(job, _run_v2_excel_generation, request.analysis_id)`.
- Return `{"job_id": job.id, "status": "started"}`.

3. Add background generation function `_run_v2_excel_generation(analysis_id: str) -> dict`:
- Import `generate_v2_excel` from `v2.output.excel_generator` (lazy).
- Look up stored results from `_analysis_results[analysis_id]`.
- **Generate Executive Summary via Claude Sonnet API call:**
  - Use sync `anthropic.Anthropic().messages.parse()` (this runs in a background thread, NOT async).
  - Define `ExecutiveSummaryResponse(BaseModel)` with `gesamtbewertung: str` and `empfehlungen: list[str]`.
  - Build a German prompt summarizing: total positions, match counts by status, top critical gaps, dimension breakdown.
  - Keep prompt under 2000 tokens. Use model `claude-sonnet-4-20250514`.
  - System prompt: "Du bist ein technischer Berater fuer Tueren und Brandschutz bei der Frank Tueren AG. Erstelle eine professionelle Zusammenfassung der Machbarkeitsanalyse auf Deutsch. Die Zusammenfassung ist fuer das Vertriebsteam und wird an Kunden weitergeleitet."
  - Wrap in try/except -- if Claude API fails, use a fallback summary: statistics only, no AI text. Log the error.
- Call `generate_v2_excel(positions, match_results, adversarial_results, gap_reports, ai_summary, ai_recommendations)`.
- Cache bytes: `offer_cache.set(f"v2_result_{analysis_id}_xlsx", xlsx_bytes, ttl_seconds=3600)`.
- Return `{"result_id": analysis_id, "has_result": True}`.

4. Add GET `/api/offer/{result_id}/download` endpoint:
```python
@router.get("/offer/{result_id}/download")
async def v2_download_result(result_id: str):
```
- Check `offer_cache.get(f"v2_result_{result_id}_xlsx")`.
- If None, return 410 with German error message.
- Return `Response(content=data, media_type="application/vnd.openxmlformats-officedocument.spreadsheetml.sheet")`.
- Filename header: `Machbarkeitsanalyse_{date}_{result_id}.xlsx` where date is `datetime.now().strftime("%Y%m%d")`.
